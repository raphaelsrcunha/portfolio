My Projects

Quebec Med

An API developed in Java with Spring for clinic management.

This is a project in which I developed a REST API in Java with the Spring framework for clinic management. It allows you to create, read, edit and delete doctor, patient and medical consultation entities. To do this, you need to have created a user to make calls via the API with an e-mail address and password. Access was built and managed with Spring Security, using a JWT token. The records are stored in the MySQL database. The interface between the application and the database was built using JPA. Flyway was used to version the database. Lombok was used to avoid boilerplate code. The various business rules that make sense for the application in question, which can be seen on the project's Trello board, are validated in the domain layer. The application was built following good practices and with a folder architecture to facilitate code maintenance and application evolution.

The backlog for our application is: implement more unit and integrated tests and develop the application's frontend.
